# Toro Solutions Website

Official website for Toro Solutions, a software company specializing in crafting innovative software solutions.

## Tech Stack
- **Framework:** [React](https://reactjs.org/) (TypeScript)
- **Bundler:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```

### Build
Build for production:
```bash
npm run build
```
